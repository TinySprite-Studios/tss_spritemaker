# Quick Start

## For Users (Using Pre-built Release)

1. Go to [Releases](https://github.com/yourusername/tss-spritemaker/releases)
2. Download the installer for your OS:
   - **Windows**: `TSS-Sprite-Maker-Setup-x.x.x.exe`
   - **macOS**: `TSS-Sprite-Maker-x.x.x.dmg`
   - **Linux**: `TSS-Sprite-Maker-x.x.x.AppImage`
3. Run the installer
4. Launch TSS Sprite Maker
5. Start drawing!

## For Developers

```bash
# Clone the repository
git clone https://github.com/yourusername/tss-spritemaker.git
cd tss-spritemaker

# Install dependencies
npm install

# Start development
npm start

# Build for release
npm run build
```

## Create Your First Release

```bash
# Make changes, commit, then create a tag:
git tag -a v0.1.0 -m "Initial release"
git push origin v0.1.0

# GitHub will automatically build and create a release!
# Check Actions tab to monitor the build
# Check Releases tab when complete
```

## Features

✏️ **Draw** - Smooth drawing with customizable brush  
🎨 **Customize** - Change brush size and color  
🧹 **Erase** - Remove parts of your artwork  
💾 **Export** - Save as PNG sprite  
🚀 **Open Source** - Free to use and modify
